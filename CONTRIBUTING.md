We welcome contributions to [Project Name]. 

## Legal: No Copyright Assignment (Inbound=Outbound)
**We do not require a Contributor License Agreement (CLA).** You retain full copyright ownership of your code.

By submitting a pull request or patch to this project, you agree to license your contribution under the **Total Reciprocity Public License (TRPL) v1.0** (or any later version). You certify that you have the right to submit this code under these terms.

## The "Signed-off-by" Requirement (DCO)
To confirm that you wrote the code or have the right to pass it on as an open-source contribution, we ask that you sign off your commits. This certifies the following **Developer Certificate of Origin**:

> "I certify that I wrote this contribution, or I have the right to submit it as an open-source contribution under the TRPL v1.0."

You can sign off your commit automatically with git:
`git commit -s -m "Your commit message"`

## Guidelines
1.  **License Compatibility:** Do not copy-paste code from StackOverflow or other projects unless they are compatible with the TRPL (Public Domain or strong copyleft).
2.  **Functional Output:** If your contribution changes how the software generates output (e.g., a new compiler backend), ensure you understand that this expands the viral scope of the so
ftware.
